# ⚙️ Predictive Maintenance AI Platform

An industrial-grade AI system that predicts machine failures and estimates remaining useful life using machine learning.

## 🚀 Features

- Machine Failure Prediction (Classification)
- Remaining Useful Life Prediction (Regression)
- Real-Time Sensor Simulation
- Interactive Dashboard (Streamlit)
- 3D Data Visualization
- Anomaly Detection System
- Alert System (Visual + Sound)

## 🧠 Models Used

- Random Forest Classifier
- Random Forest Regressor
- Linear Regression (Trend Analysis)

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Plotly

## 📊 How It Works

1. Data preprocessing
2. Model training
3. Model serialization (.pkl)
4. Real-time dashboard predictions
